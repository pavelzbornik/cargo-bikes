# Tail Hitch L

The Tern Tail Hitch L is a brand-specific trailer coupling designed exclusively for Tern cargo bikes. It provides a secure and standardised hitch point at the rear of the bike, enabling riders to tow a compatible single-arm or two-arm bike trailer. The "L" designation refers to the long-tail frame geometry of the Tern GSD platform.

## Features

- Standardised trailer coupling interface compatible with common trailer hitch systems (e.g., single-arm trailers)
- Engineered specifically for the Tern GSD rear frame geometry
- Robust steel construction for reliable load transfer
- Quick-attach and detach mechanism for convenient trailer connection
- Low mounting position keeps trailer stable and centre of gravity low
- Designed to work alongside existing Tern rear rack and cargo systems

## Compatibility

This accessory is designed **specifically for Tern GSD long-tail cargo bikes**. It is not a universal hitch and is not intended for use on third-party frames. Installation requires the Tern GSD's dedicated hitch mounting points.
