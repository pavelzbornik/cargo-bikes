# Bosch Performance Sport Gen 3

Mid-drive motor by [[Bosch]], part of the Bosch Smart System ecosystem. Positioned as the performance-oriented fitness and trekking motor — a quieter, lighter alternative to the [[Bosch Performance CX]] for road, urban, and cargo applications that do not require maximum trail torque.

## Compatibility

- **Drivetrains:** Derailleur (7–12 speed), internally geared hubs (Shimano Nexus, Alfine)
- **Belt drive:** Compatible with Gates Carbon Drive
- **Chainring interface:** 104 BCD
- **eShift:** Compatible with Shimano Di2 and SRAM eTap
- **Displays:** Bosch Kiox 300, Kiox 500, Nyon, Intuvia 100
- **System:** Bosch Smart System only — not backwards compatible with Legacy System components

## Features

- Walk assist mode
- Bluetooth connectivity via Bosch Flow app (ride mode customization, OTA firmware updates)
- Narrower Q-factor than Performance CX, suited to road and urban bike geometry
- Refined noise dampening vs Gen 2
- Smooth, linear assist feel tuned for trekking and fitness riding
