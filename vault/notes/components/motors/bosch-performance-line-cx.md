# Bosch Performance Line CX

The Bosch Performance Line CX is a mid-drive motor produced by [[Bosch]] and is the dominant powertrain choice in the European cargo e-bike segment. First introduced in 2018, the current generation runs on Bosch's Smart System platform, launched in 2022. It is specifically engineered for high-torque, high-load applications such as cargo bikes, longtails, and fully-loaded utility e-bikes.

## Features

- **eMTB mode** — dynamically adjusts support level in response to terrain and pedaling input; widely used in cargo applications for varied urban terrain
- **Extended Boost** — brief burst of additional power from a standstill, suited to loaded cargo starts
- **Walk Assist** — motor-assisted walking mode at low speed
- **Freewheel mechanism** — motor decouples above assisted speed, reducing drag and noise when unassisted
- **Smart System connectivity** — Bluetooth and ANT+, OTA firmware updates via Bosch eBike Flow app, compatible with Kiox 300, Kiox 500, and Nyon displays
- **Motor lock** — anti-theft lock function via the Smart System app
- **Reduced noise** — redesigned internal gearing and improved damping vs. earlier CX generations

## Mounting

The Performance Line CX uses Bosch's proprietary integrated motor housing, which replaces the bottom bracket assembly. Frames must be designed specifically for the Bosch CX mount — it is not a universal aftermarket fit. Compatible with BSA (English threaded) and press-fit BB shell interfaces depending on frame design.

## Market Position

The Performance Line CX sits at the top of [[Bosch]]'s cargo and utility motor lineup. It is an OEM-only component — not sold retail as a standalone unit. As a system contribution to bike pricing, models equipped with the CX typically start around €3,000–€4,000 for base utility builds and reach €8,000–€12,000+ for fully-configured cargo platforms with dual batteries and premium component builds.
